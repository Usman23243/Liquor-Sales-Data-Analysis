# 🍷 Liquor Sales Data Analysis (End-to-End Project)

## 📌 Project Overview
Is project mein maine ek massive Liquor Sales dataset (takreeban 1.2 Crore+ rows) ka end-to-end data analysis kiya hai. Mera maqsad is bare data se aham business insights nikalna tha.

## 🛠️ Tech Stack Used
* **Python:** Data extraction aur scripting.
* **Pandas:** Data manipulation aur chunking.
* **SQLite / SQLAlchemy:** Massive dataset ko database mein load karna aur SQL queries likhna.
* **Matplotlib:** Data visualization (Graphs).

## 🚀 Key Insights (Business Findings)
1. **Top Brand:** Jack Daniels No 7 Black ne sab se zyada revenue generate kiya (~$8 Million).
2. **Top Store:** Store 76 sab se behtareen branch sabit hui jisne ~$25 Million ki sales ki.
3. **Best Month:** December mein sab se zyada sales hui ($52.3M), jabke July mein bhi unexpectedly badi spike aayi.
